SEO Automation Platform — Full-stack app automating SEO content creation. Features: 6-step workflow (platform selection, keyword research, SERP scraping, AI-generated titles/descriptions, section content generation, JSON publishing), multi-language translation with GitHub sync, exact match domain (EMD) management with website generation, JSON-to-website conversion, and settings for platforms/sections/templates. Built with Next.js, Fastify, TypeScript, Gemini AI, and Firebase.

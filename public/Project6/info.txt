AI Comic Book Generator with Interactive Story Viewer
Project Overview
Comic Book Generator is an AI-powered Python application that transforms real photos of people into a personalized, illustrated children’s storybook. The system generates both a cohesive story and matching illustrations while preserving character identity and visual consistency across all pages. The final output can be viewed in a polished, interactive flipbook-style story viewer.

Core Capabilities
AI Story & Illustration Generation
The generation engine takes photos of a father, mother, and child as reference inputs and uses a multimodal AI model to preserve facial features, hairstyles, and overall character appearance. It creates a custom 10-part children’s story based on a user-defined theme, ensuring the narrative is wholesome, sequential, and age-appropriate.

To maintain a consistent art style, the system first generates a cover image in the selected visual style. This cover image is then used as a reference for all subsequent chapters, ensuring uniform illustration style throughout the book. The complete story, including text and base64-encoded images, is saved in a structured JSON format.

Interactive Flipbook Viewer
The project includes a web-based story viewer that presents the generated content as a realistic digital book. It features smooth page-flip animations, a split layout with illustrations on one side and story text on the other, and theme toggles for day and night reading modes. The viewer can load stories directly from the generated JSON file or from a local image folder.

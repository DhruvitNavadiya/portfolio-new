ComfyUI Face-Blend Baby Generator
Project Overview
This project is a custom ComfyUI workflow designed to generate ultra-realistic baby images by blending facial features from two parent photos. The workflow focuses on identity preservation, skin-tone balance, and age realism, producing high-quality, photorealistic outputs suitable for repeatable generation.

The entire pipeline is built as a reusable, production-ready ComfyUI graph rather than a simple text-to-image setup.

What the Workflow Does
Takes two input images (parent faces)
Extracts and preserves key facial identity features
Blends visual traits naturally into a baby-like face
Enforces realistic baby proportions, skin texture, and lighting
Filters out adult features, artifacts, and stylized outputs
Produces consistent, high-quality results with controlled sampling
Core Components & Logic
Identity Preservation
Uses IPAdapter FaceID with InsightFace to retain facial structure and key identity traits from both parent images.
Dual FaceID nodes allow balanced influence from each input face.
Prompt Control & Conditioning
CLIP text conditioning is carefully tuned to enforce:
Baby age realism
Soft skin, youthful facial structure, and natural expressions
A Prompt Replace node dynamically adjusts prompts based on detected attributes, ensuring consistent tone and realism.
Strong negative prompts are applied to remove adult features, artifacts, and unwanted styles.
Skin Tone & Visual Balance
Integrated skin-tone detection logic helps maintain natural blending between parent images.
Ensures even skin tones and avoids unnatural color shifts.
Image Generation Pipeline
Built on SDXL with controlled sampling parameters.
Uses fixed seeds and tuned schedulers for repeatable results.
Outputs are decoded, previewed, and validated directly within the workflow.
Technical Highlights
Modular ComfyUI graph design
IPAdapter FaceID + InsightFace integration
CLIP Vision conditioning
Prompt replacement and negative filtering
SDXL-based photorealistic sampling
Real-time preview and iteration support
